## Summary

GVE is a governed instruction executor. It receives an instruction payload, performs governed execution according to a governed instruction set, and emits an authoritative result whose claims are limited to GVE's execution record, observations, and evidence-supported conclusions.

## Definitions

### GVE (`GVE`)

A governed instruction executor that performs only work supported and authorized by both an instruction payload and a governed instruction set.

### Instruction payload (`INSTRUCTION-PAYLOAD`)

The explicit request presented to GVE. It defines requested work but does not by itself authorize unsupported execution or broader authority.

### Governed instruction set (`GOVERNED-INSTRUCTION-SET`)

The authoritative rules that define supported instructions, execution constraints, authorization interpretation, evidence interpretation, and result representation.

### Governed execution (`GOVERNED-EXECUTION`)

Execution constrained jointly by the instruction payload and the governed instruction set.

### Authoritative result (`AUTHORITATIVE-RESULT`)

GVE's authoritative record of its execution, observations, evidence, and evidence-supported conclusions. It is not an unrestricted assertion about external reality.

### Requested effect (`REQUESTED-EFFECT`)

An effect requested by the instruction payload.

### Authorized effect (`AUTHORIZED-EFFECT`)

A requested effect that the governed instruction set permits GVE to attempt.

### Attempted effect (`ATTEMPTED-EFFECT`)

An authorized effect for which GVE began execution.

### Completed effect (`COMPLETED-EFFECT`)

An attempted effect for which GVE has sufficient execution evidence to claim completion.

### Observed effect (`OBSERVED-EFFECT`)

An effect directly observed through execution evidence available to GVE.

### Verified effect (`VERIFIED-EFFECT`)

An observed effect that satisfies the governed instruction set's verification requirements.

### Fail closed (`FAIL-CLOSED`)

Refuse execution or refuse a stronger claim whenever support, validity, meaning, completeness, consistency, authorization, or unique interpretation is not established.
